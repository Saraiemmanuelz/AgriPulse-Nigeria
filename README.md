# 🌾 AgriPulse Nigeria — Agricultural Intelligence Dashboard

> **Data-Driven Decisions. Stronger Farms. Food Secure Nigeria.**

**Project Type:** Business Intelligence Case Study
**Status:** Completed Portfolio Project

An end-to-end Power BI analytics solution built using Nigeria Agricultural Survey (NASS) data to uncover insights into crop performance, seasonal productivity, farmer demographics, and agricultural technology adoption.

---

## 📌 Project Overview

AgriPulse Nigeria is a five-page interactive Power BI dashboard designed to transform agricultural survey data into actionable insights for policymakers, agribusiness stakeholders, researchers, and development institutions.

This project demonstrates applied skills across the complete analytics workflow:

✔ Data Cleaning
✔ Data Modeling
✔ DAX Development
✔ Dashboard Design
✔ Data Storytelling
✔ Insight Generation

### Objectives

* Measure agricultural productivity across Nigeria
* Evaluate crop profitability and yield efficiency
* Compare seasonal agricultural performance
* Analyze farmer demographics and technology adoption
* Support evidence-based agricultural decision-making

# 🔍 Key Findings

## 🌱 Crop Insights

* Cassava recorded the highest production output.
* Yam achieved the strongest profitability.

## 📍 Geographic Insights

* North West generated the largest agricultural revenue.
* Yield efficiency varied across regions.

## 🌦 Seasonal Insights

* Main Season significantly outperformed Dry Season.

## 👩‍🌾 Household Insights

* Female-led households represented a substantial share.
* Improved seed adoption correlated with higher income.
